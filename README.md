# Multi-Engine-Fraud-scanner

## Overview

Multi-Engine Cybersecurity Scanner is an MVP designed to help everyday users quickly assess potential cyber threats through a single interface.

The idea originated from observing the growing number of cyber fraud incidents in India and noticing that many people either do not use security tools at all or rely on a single solution such as a VPN or password manager. Security tools are often scattered across different platforms, making security checks time-consuming and difficult for non-technical users.

This project centralizes multiple security tools into one workflow, allowing users to submit suspicious URLs, emails, phone numbers, etc and receive consolidated scan results within seconds.


## How It Works

1. User submits a suspicious URL.
2. The workflow validates the input.
3. The URL is analyzed using multiple security services.
4. Results are aggregated into a single response.
5. Users receive a quicker and more comprehensive threat assessment.

## Why n8n?

The project was intentionally built using n8n to rapidly develop and validate the idea as an MVP before investing in a full-scale application. This allowed quick experimentation, workflow automation, and user feedback collection.
